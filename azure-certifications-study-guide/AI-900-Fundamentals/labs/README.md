## Prerequisites

Before starting the labs:

1. **Azure Account**: You can use the Microsoft Learn sandbox (free) or your own Azure subscription
2. **Web Browser**: Modern browser (Edge, Chrome, Firefox)
3. **Basic Computer Skills**: Familiarity with web interfaces

## Using Microsoft Learn Sandbox

Most labs can be completed using the free Microsoft Learn sandbox:

1. Navigate to the lab exercise on Microsoft Learn
2. Click "Activate sandbox"
3. Wait for sandbox to provision (1-2 minutes)
4. Follow the lab instructions

**Note**: Sandbox sessions last 4 hours maximum.
