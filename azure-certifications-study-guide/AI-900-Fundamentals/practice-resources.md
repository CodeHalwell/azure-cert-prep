## Exam Preparation Checklist

### 1 Week Before Exam
- [ ] Complete all practice assessments
- [ ] Review incorrect answers
- [ ] Re-read key concepts
- [ ] Take notes on weak areas

### 2-3 Days Before Exam
- [ ] Final practice assessment
- [ ] Review Responsible AI principles
- [ ] Review Azure AI services overview
- [ ] Get familiar with exam sandbox

### Day Before Exam
- [ ] Light review only
- [ ] Verify exam appointment
- [ ] Prepare ID and testing space
- [ ] Get good sleep

### Exam Day
- [ ] Arrive/login 15 minutes early
- [ ] Stay calm and focused
- [ ] Read each question carefully
- [ ] Flag uncertain questions for review
