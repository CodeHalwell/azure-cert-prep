## üìö Domain Details

### Domain 1: Develop Azure Compute Solutions (25-30%)

| Topic | Key Concepts |
|-------|--------------|
| **Azure App Service** | Web apps, deployment slots, configuration |
| **Azure Functions** | Triggers, bindings, Durable Functions |
| **Azure Container Apps** | Container deployment, scaling |
| **Azure Container Registry** | Image management |

### Domain 2: Develop for Azure Storage (15-20%)

| Topic | Key Concepts |
|-------|--------------|
| **Blob Storage** | Containers, tiers, lifecycle |
| **Cosmos DB** | APIs, partitioning, consistency |
| **Storage SDKs** | .NET, Python, JavaScript |

### Domain 3: Implement Azure Security (20-25%)

| Topic | Key Concepts |
|-------|--------------|
| **Authentication** | Microsoft Identity, MSAL |
| **Authorization** | RBAC, SAS, stored policies |
| **Secure data** | Encryption, Key Vault |
| **Managed Identities** | System/user-assigned |

### Domain 4: Monitor, Troubleshoot, Optimize (15-20%)

| Topic | Key Concepts |
|-------|--------------|
| **Application Insights** | Telemetry, availability tests |
| **Azure Cache for Redis** | Caching patterns |
| **Azure CDN** | Content delivery |

### Domain 5: Connect to and Consume Services (15-20%)

| Topic | Key Concepts |
|-------|--------------|
| **API Management** | Policies, versions, products |
| **Event Grid** | Event routing |
| **Service Bus** | Queues, topics, subscriptions |
| **Azure Queue Storage** | Message queuing |

## üí° Exam Tips

1. **Know the SDKs** - .NET, Python, JavaScript code patterns
2. **Understand triggers and bindings** - Azure Functions
3. **Practice with containers** - Container Apps, ACI
4. **Know authentication flows** - MSAL, managed identities
5. **Understand messaging** - Service Bus vs Event Grid vs Queues
