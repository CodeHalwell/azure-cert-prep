### What is an AI Agent?

An AI agent is an autonomous system that:
1. **Perceives** its environment through inputs
2. **Reasons** about its goals and constraints
3. **Acts** by executing tools and functions
4. **Learns** from feedback and outcomes

### Model Context Protocol (MCP)

MCP provides a standardized way for AI agents to:
- Access external tools and APIs
- Manage context across conversations
- Share information between agents
- Maintain security and governance
