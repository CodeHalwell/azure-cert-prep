## ‚úçÔ∏è Exam-Taking Strategies

### General Strategies

| Strategy | Description |
|----------|-------------|
| **Read carefully** | Don't rush, understand the question |
| **Look for keywords** | "MOST," "BEST," "FIRST," "LEAST" |
| **Eliminate wrong answers** | Narrow down choices |
| **Flag difficult questions** | Return after finishing |
| **Trust your preparation** | First instinct often correct |

### Time Management

| Approach | Details |
|----------|---------|
| **Calculate pace** | Total time / questions = time per question |
| **Set checkpoints** | Quarter, half, three-quarter marks |
| **Don't get stuck** | Flag and move on |
| **Save time** | For review at the end |

### Question Types

| Type | Strategy |
|------|----------|
| **Multiple choice** | Read all options before choosing |
| **Multiple select** | Note how many to select |
| **Drag and drop** | Understand the sequence or matching |
| **Case study** | Read scenario carefully, take notes |
| **Hot area** | Click precisely on correct area |

## ‚ùå Common Mistakes to Avoid

### Study Mistakes

| Mistake | Better Approach |
|---------|-----------------|
| Only watching videos | Do hands-on labs |
| Cramming last minute | Steady, spaced study |
| Skipping practice tests | Take many practice exams |
| Ignoring weak areas | Focus extra time there |
| Not using Azure | Get hands-on experience |

### Exam Mistakes

| Mistake | Better Approach |
|---------|-----------------|
| Not reading fully | Read entire question |
| Rushing | Use all available time |
| Changing answers | Trust first instinct |
| Leaving blank | Always answer something |
| Panicking | Take deep breaths, stay calm |
